# StS2-mod-IsekaiHero
This mod adds the Isekai Hero playable character to Slay the Spire 2.

## Design Goal

The Isekai Hero is a genre-parody power-fantasy character with outsider logic:
they gain a **Job**, find ways to **Exploit** the Spire's systems, and earn
absurd **Cheat Skills** without turning every combat into an automatic win.

The current character goal and design filters live in
[docs/isekai-hero-goal.md](docs/isekai-hero-goal.md).
